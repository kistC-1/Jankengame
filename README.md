Jankengame
==================
コンピュータとじゃんけんをするゲームです。


### 操作の仕方 ###
------
スタートボタンを押すとじゃんけんが始まります。  
じゃんけんがはじまると、コンピュータ（相手）の画面が動きます。  
「グー・チョキ・パー」のボタンを押すと、じゃんけんが行われます。  
もう一度ジャンケンを行うときは再度スタートボタンを押してください。
